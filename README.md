# TicTacToeWPF

# Архитектура:
Модель: GameState хранит состояние игры.
Представление: XAML-разметка и стили.
Логика: Обработчики событий в MainWindow.xaml.cs.
# Установка:
Установите .NET Core SDK.
Клонируйте репозиторий.
Откройте решение в Visual Studio.
Восстановите пакеты NuGet.
Запустите приложение.
# Функции:
Новая игра: Меню → Игра → Новая игра.
Сохранение/загрузка: Используйте соответствующие пункты меню.
Смена темы: Меню → Тема.
# Запустите приложение
Нажмите F5 в Visual Studio для запуска.
Играйте, используя мышку для выбора клеток.
